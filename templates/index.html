Make Those Playlists Smooth Like Buttah.

The Camelotinator tool is a simple web app designed to make the song transitions seamless in all your favorite Spotify playlists. Using Spotify's own web API and some incredible open-source tools and libraries, Camelotinator is designed to give you an easy and effective playlist arrangment that greatly enhances Spotify's new Mix feature. Interested? Then what are you waiting for? Head on down to the Camelotify button below to get started! Note: you will have to login to Spotify so Camelotinator can see (and if you prefer, copy) and re-arrange your playlists.
